# MERGE SORT

## '정확히 반절로 나눠서 정렬한다'

퀵 정렬처럼 '분할 정복' 방법을 채택한 알고리즘이다.
퀵 정렬은 편향되게 분할할 가능성이 있어서 최악의 경우 O(N^2)의 시간복잡도를 가질수 있지만
병합정렬은 정확히 반으로 나누어 정렬하는 방식으로 최악의 경우에도 O(NlogN)을 보장한다.
